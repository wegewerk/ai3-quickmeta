======
Editor
======

Ai3 Quickmeta adds an AI-assisted metadata editor to the page properties. It
helps you generate page titles, descriptions, and other metadata based on the
actual content of the page.

.. _editor-access:

-------------------------
Accessing the Assistant
-------------------------

1.  Open the **Page Properties** of the page you want to edit.
2.  Navigate to the **Quickmeta** tab.

.. _editor-workflow:

--------
Workflow
--------

1.  **Generate Suggestions**: Click the button **"Generate metadata"**.
    The assistant will analyze the page content and provide suggestions for
    various fields.
2.  **Review Suggestions**: The suggested values will appear in the right column.
    You can manually edit these suggestions if needed.
3.  **Transfer Values**:
    *   **Transfer All**: Copies all suggestions to the corresponding TYPO3
        fields.
    *   **Transfer Empty**: Copies suggestions only to fields that are
        currently empty.
    *   **Single Field Transfer**: Click the arrow button next to a specific
        field to transfer only that value.
4.  **Save**: Click **"Save"** in the top bar of the page properties to persist
     the changes.

.. _editor-fields:

-----------------
Supported Fields
-----------------

The assistant can provide suggestions for the following fields:

*   Page Title
*   Navigation Title
*   SEO Title
*   Abstract
*   Description
*   Open Graph (Facebook) Title and Description
*   X/ Twitter Card Title and Description
