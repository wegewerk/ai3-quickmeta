=============
Ai3 Quickmeta
=============

:Extension key:

:Package name:

:Version:

----

Ai3 Quickmeta is a TYPO3 extension that provides an AI-powered assistant for
generating page metadata (title, description, keywords) using the **ZAK-AI REST API**. It is part
of the *Ai3 Suite* and requires :composer:`wegewerk/ai3_core`..
